# 🌿 Agropecuária Catarinense — Web App

Aplicação web interativa sobre a agropecuária de Santa Catarina.
Desenvolvida como trabalho de Geografia — 2.° Ano do Ensino Médio.

---

## 🚀 Como abrir

Basta abrir o arquivo `index.html` diretamente no navegador.
Não precisa de servidor, instalação ou internet (apenas as fontes do Google Fonts carregam online).

---

## 📂 Estrutura de pastas

```
agropecuaria-sc/
├── index.html              ← Arquivo principal (abra esse)
├── css/
│   └── style.css           ← Todos os estilos e animações
├── js/
│   └── main.js             ← Todas as interações e efeitos
├── assets/
│   └── images/             ← Coloque suas imagens aqui (veja lista abaixo)
└── README.md               ← Este arquivo
```

---

## 🖼️ Imagens — Como adicionar

Coloque suas fotos na pasta `assets/images/` com os nomes exatos abaixo:

| Nome do arquivo       | O que deve mostrar                          |
|-----------------------|---------------------------------------------|
| `hero-campo.jpg`      | Vista aérea de lavoura ou paisagem rural    |
| `suinos.jpg`          | Granja / criação de suínos                  |
| `frango.jpg`          | Aviário / frango de corte                   |
| `maca.jpg`            | Pomar ou colheita de maçã                   |
| `cebola.jpg`          | Lavoura ou colheita de cebola               |
| `arroz.jpg`           | Lavoura de arroz irrigado                   |
| `leite.jpg`           | Pecuária leiteira / vaca                    |
| `familia-rural.jpg`   | Família de agricultores                     |
| `colheita-maca.jpg`   | Pessoa colhendo maçã no pomar               |
| `drone-lavoura.jpg`   | Drone sobrevoando lavoura                   |
| `frigorifico.jpg`     | Interior de frigorífico                     |
| `lavoura-arroz.jpg`   | Lavoura de arroz vista de cima              |
| `cooperativa.jpg`     | Caminhões na cooperativa agrícola           |

> 💡 **Dica:** Baixe imagens de alta qualidade em sites como
> [Unsplash](https://unsplash.com), [Pexels](https://pexels.com) ou
> [Pixabay](https://pixabay.com) buscando termos como:
> "pig farm Brazil", "apple orchard harvest", "rice field aerial", etc.
> As imagens funcionam mesmo sem adicionar (o site mostra o emoji de fallback).

## ✨ Funcionalidades do site

- **Partículas animadas** flutuando no hero (grãos e sementes)
- **Cursor personalizado** com efeito de seguimento suave
- **Contador animado** nas estatísticas (os números sobem ao rolar)
- **Mapa interativo** de SC — clique nas regiões para ver detalhes
- **Cards 3D** com efeito tilt (inclinação ao mover o mouse)
- **Barras de progresso** animadas nos produtos
- **Linha do tempo** histórica interativa
- **Galeria com lightbox** (clique nas imagens para ampliar)
- **Loader** com animação de entrada
- **Navbar** que muda ao rolar + menu mobile
- **Animações de scroll** em todos os elementos
- **Responsivo** para celular, tablet e desktop
